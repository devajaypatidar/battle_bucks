API Endpoints Overview

POST /api/v1/auth/register User Registration
POST /api/v1/auth/login User Login
GET /api/v1/store/items Browse Store Items
POST /api/v1/purchases Create Purchase
GET /api/v1/inventory Get User Inventory
GET /api/v1/characters Get Character Profiles
POST /api/v1/inventory/:itemId/use Use Consumable Item

Quick Start Guide

  1. Start the BattleBucks API server: npm run start:dev
  2. Open the client application by clicking "Launch Application"
  3. Register a new account or use the demo credentials
  4. Explore all features: store, inventory, characters, purchases
  5. Check the API documentation for technical details
